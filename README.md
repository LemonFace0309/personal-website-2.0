## 🛠 Wanna run on your machine?

1. Clone the project

   ```sh
   git clone https://github.com/LemonFace0309/personal-website-2.0
   ```

2. Install the dependencies

   ```sh
   cd personal-website-2.0
   npm i
   ```

3. Start the development server

   ```sh
   npm run dev
   ```
